[Notes]
- This is a simple resistive divider intended for no-load operation.
- With Vin = 5.0V and R1 = R2 = 10k, the nominal output is 2.50V.
- Use 1% resistors or better to keep VOUT within 2.50V ±0.05V under worst-case resistor tolerance.
- If a load will be connected to VOUT_2V5, add a buffer (for example, an op-amp voltage follower) or use a dedicated 2.5V reference.
- Place R1 and R2 physically close together to improve matching and reduce noise pickup at the divider node.
